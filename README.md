# TFG_Miguel_Laloma

Este proyecto configura un entorno de desarrollo local para WordPress utilizando Docker y Docker Compose.

## Requisitos

- [Docker](https://www.docker.com/get-started)
- [Docker Compose](https://docs.docker.com/compose/install/)
